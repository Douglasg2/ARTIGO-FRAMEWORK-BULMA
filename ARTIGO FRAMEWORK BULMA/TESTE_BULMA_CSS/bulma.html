Bulma Framework

Bulma é uma biblioteca CSS. Isso significa que ele fornece classes. CSS para ajudá-lo a estilizar seu código HTML. Para usar o Bulma, você pode usar o arquivo .css pré-compilado ou instalar os arquivos .sass para personalizá-lo de acordo com suas necessidades.

Sobre

O Bulma é um framework open-source CSS baseado em Flexbox criado por Jeremy Thomas. O Bulma tem semelhanças ao Framework Bootstrap, o qual além de semelhante, foi utilizado como base.

Elementos

O Framework contém todos os elementos mais comuns, como: formulários, tabelas, botões. menus, títulos, notificações, barras de progresso, entre outras coisas – assim como o consagrado Bootstrap.

Integração com o FontAwesome.

Icone bonitos e estilosos que podem ser usados sem muito esforço.

Como Importar

Para Importar o Bulma em seu arquivo CSS externo, basta importar a seguinte url:
@import "https://cdn.jsdelivr.net/npm/bulma@0.9.4/css/bulma.min.css";

Para Importar o Bulma em seu arquivo Html, basta adicionar a tag link com a url:
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.4/css/bulma.min.css">

Vantagens

  • 100% responsivo (mobile first)

  • Totalmente CSS (não requer JavaScript)

  • Modular (importe no seu projeto somente o que você precisa)

  • Moderno (construído com Flexbox)

  • Livre (open Source no GitHub)
